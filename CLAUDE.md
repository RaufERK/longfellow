# Longfellow App

Маркетплейс книг и печатной продукции на Next.js App Router с редактируемым каталогом, корзиной и оформлением заказов.

## Цели

Быстрое отображение каталога, поиск, корзина, оформление заказов и простая модерация ассортимента через `/moderator`.

## Стек

Next.js 16 (App Router), React 19, TypeScript, Tailwind/shadcn-ui, Prisma + SQLite, Redis, PM2 для прод-запуска, Turbopack в dev, Webpack в build.

## Директории

`src/app` — страницы и API-роуты.  
`src/components` — общие UI-блоки (единое меню).  
`src/lib` — утилиты, Prisma/Redis, валидация.  
`prisma` — схема, миграции.  
`products` — подготовленные JSONы каталога.  
`public` — статика и изображения товаров.

## Правила кодирования

Только функциональные компоненты и хуки, пропсы деструктурировать сразу в сигнатуре.  
Без классов, сервис-слоёв, тестов и лишних абстракций.  
Общий навбар переиспользуется на всех страницах.  
Минимальный размер шрифта 18px, базовый шрифт `Times, "Times New Roman", serif`.  
Если нужен админ, размещаем под `/moderator`.

## Архитектура

Данные через Prisma из SQLite; для фоновых задач — Redis.  
API расположены в `app/api/*`; фронт использует серверные/клиентские компоненты без серверных Actionов.  
Деплой через PM2 `ecosystem.config.cjs`, миграции выполняются в пост-деплой скрипте.

## Именование

Компоненты — PascalCase, функции и переменные — camelCase, файлы с компонентами — PascalCase или page.tsx в маршрутах.
